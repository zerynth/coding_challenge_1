# Under the heap

Write a memory allocator in C trying to minimize memory fragmentation.

---
## Instructions

1. Edit `CMakeLists.txt` to adjust the pre-compile macros. More details about
each macro can be found in `mem_alloc.h`.
2. Compile demo - `make compile`
3. Run demo - `make run`
4. Clean - `make clean`

**Note:** Cmake is necessary.
